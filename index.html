About this project

This project has been made as a way for me to learn basic HTML, CSS and JavaScript. I've always heard the best way to learn how to code is by making personal projects. I'm a very amateur pizza maker for my family so I thought this project would be perfect to keep myself motivated.

There are a million things I could improve about this site. A feature I'd like to add some time in the future would be the option for user generated lists, either saving them with local storage or learning how to work with cookies.

The way to use this site is quite simple, click the roll buttons until you're satisfied, then show your finished product.
